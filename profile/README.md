## 🚀 What is Ploi?

[Ploi](https://ploi.io) is a powerful server management platform that provisions, configures, and deploys your applications with minimal effort. Connect a cloud provider, link your Git repository, and deploy — Ploi handles the rest.

> "Ploi truly allows you to focus on building apps rather than configuring servers. Huge time and cost saver!" — *Lars Klopstra, Founder of Flowframe*

---

## ✨ Features

| Category | Details |
|---|---|
| **1-Click Installs** | Laravel, WordPress, Statamic, Craft CMS, Nextcloud, phpMyAdmin |
| **Server Providers** | UpCloud, DigitalOcean, Linode, Hetzner, AWS EC2, Custom VPS |
| **Deployments** | Zero-downtime deploys, Git integration, automated pipelines |
| **Stack** | NGINX, PHP, Redis installed by default |
| **SSL** | Free Let's Encrypt certificates, 1-click install |
| **Databases** | Automated database & file backups |
| **Networking** | Load balancing, DNS management, website isolation |
| **Monitoring** | Server monitoring, site monitoring, server insights |
| **Notifications** | Email, Slack, Discord, Telegram, mobile push |
| **Teams** | Team management, collaboration, whitelabel support |
| **Mobile** | iOS & Android app for managing servers on the go |

## 🛠️ Get Started in 3 Steps

```
1. Create a server     →  Select UpCloud, DigitalOcean, Hetzner, AWS EC2, or any custom VPS
2. Connect your repo   →  Link your GitHub / GitLab / Bitbucket repository
3. Deploy              →  Click deploy, sit back ☕
```
